# Project of Research

This repository is organized for research code, literature notes, experiments,
and paper writing.

## Structure

- `docs/`: paper drafts, notes, and reference summaries
- `literature/`: key PDFs or exported papers
- `src/`: models, data pipelines, training, and evaluation code
- `experiments/`: experiment folders and experiment log
- `baselines/`: reproduced baseline implementations
- `configs/`: YAML/JSON config files
- `scripts/`: helper scripts for setup, download, and tooling
- `assets/`: figures and visualization outputs
